# Сервис Сокращения Ссылок

Простое консольное приложение на Java для сокращения URL-адресов с дополнительным функционалом.

## Функциональные возможности

- **Создание коротких ссылок**: Преобразует длинные URL в уникальные 8-символьные короткие ссылки.
- **Настраиваемый лимит переходов**: Возможность установить максимальное количество переходов для каждой ссылки.
- **Ограниченное время жизни**: Ссылки автоматически удаляются через 24 часа после создания.
- **Управление ссылками**: Команды для получения информации о ссылке и ее принудительного удаления.
- **Автоматическое открытие в браузере**: При переходе по короткой ссылке приложение автоматически пытается открыть оригинальный URL в браузере по умолчанию.

## Требования

Для сборки и запуска проекта вам понадобятся:

- Java 11 (или выше)
- Apache Maven

## Сборка и запуск

1.  Откройте терминал (командную строку) в корневой папке проекта.
2.  Выполните следующую команду:

```bash
    mvn compile exec:java -Dexec.mainClass="com.example.shortener.ConsoleApp"
```

3.  Приложение запустится в интерактивном режиме и будет ждать ваших команд.

## Руководство пользователя

После запуска приложения вы увидите приглашение к вводу `> `.

### Доступные команды

*   **Создать ссылку (с лимитом по умолчанию):**
    `create <длинный_url>`
    *Пример: `create https://www.google.com`*

*   **Создать ссылку (с указанием лимита):**
    `create <длинный_url> <лимит>`
    *Пример: `create https://yandex.ru 5`*

*   **Получить информацию о ссылке:**
    `info <короткая_ссылка>`
    *Пример: `info AbcDef12`*

*   **Удалить ссылку:**
    `delete <короткая_ссылка>`
    *Пример: `delete AbcDef12`*

*   **Перейти по ссылке:**
    Просто введите `<короткую_ссылку>`
    *Пример: `AbcDef12`*

*   **Выйти из приложения:**
    `exit`
